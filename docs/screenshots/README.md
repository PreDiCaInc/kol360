## Screenshot Guidelines

1. **Remove sensitive data** - Blur or replace real names, emails, and company info
2. **Use test data** - Screenshots should show realistic but fictional data
3. **Highlight key elements** - Consider adding red boxes around important buttons
4. **Consistent sizing** - Keep all screenshots at similar dimensions
5. **Light mode** - Use the default light theme for consistency

## How to Capture

1. Log into KOL360 as a test user
2. Navigate to each page
3. Use browser's screenshot tool or a tool like CleanShot
4. Save as PNG with the exact filename listed above
